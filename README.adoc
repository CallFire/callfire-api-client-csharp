= Callfire API v2 REST client

C# client for Callfire platform API version 2. See link:docs/GettingStarted.adoc[Getting Started]
 page for setup instructions.

.*Requirements:*
* .NET 4.5+

.*Dependencies:*
* Newtonsoft.Json >= 7.0.1
* RestSharp >= 105.2.3
